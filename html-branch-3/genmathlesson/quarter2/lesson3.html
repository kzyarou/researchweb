- **One-to-One Functions**: A function where each input corresponds to exactly one unique output, and each output corresponds to exactly one unique input. This means the function has an inverse that is also a function.

- **Real-Life Applications**: - **Cryptography**: One-to-one functions are used in encryption algorithms where each plaintext maps to a unique ciphertext, ensuring secure communication. - **Inventory Management**: Assigning unique serial numbers to products allows for precise tracking and management. - **Biometrics**: Systems like fingerprint or retina scans rely on one-to-one functions to match individuals uniquely.
